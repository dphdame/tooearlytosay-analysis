# Data Download Instructions

## Cal-ITP Transit Data

Complete statewide GTFS aggregation:
https://data.ca.gov - search "Cal-ITP GTFS"

## Individual Agency GTFS (for comparison)

Download from each agency's open data portal:
- VTA: https://www.vta.org/go/developers
- AC Transit: https://www.actransit.org/developer
- etc.
