# Data Notes

## Google Places API

This project demonstrates API collection methodology.
Due to Google ToS, raw API responses cannot be redistributed.

To replicate:
1. Get Google Cloud API key: https://console.cloud.google.com/
2. Enable Places API
3. Set GOOGLE_API_KEY in .env
4. Run collection scripts
