# Data Download Instructions

## GTFS Data

Cal-ITP statewide feed: https://data.ca.gov

## OpenStreetMap

Download California PBF file:
https://download.geofabrik.de/north-america/us/california.html

## Java Requirement

r5py requires Java 11 or higher:
```bash
java -version
```

Install via:
- macOS: `brew install openjdk@11`
- Ubuntu: `sudo apt install openjdk-11-jdk`
