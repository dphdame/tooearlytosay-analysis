# Data Download Instructions

## Crime Data

California DOJ Open Justice Portal:
https://openjustice.doj.ca.gov/data

Download: Crimes & Clearances dataset (2018-2022)

## ACS PUMS

Census Bureau Public Use Microdata Sample:
https://www.census.gov/programs-surveys/acs/microdata.html

Download California 5-year PUMS (2018-2022)
